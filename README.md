# Pictish Shadow Detector

## Project Overview
The Pictish Shadow Detector is an advanced tool designed to detect and analyze shadow patterns associated with Pictish artifacts and ancient sites. Utilizing resonance theory and post-quantum cryptography, this tool provides insights into spatial relationships and shadow coupling phenomena.

## Features
- **Live Shadow Probability Detection:** Real-time metrics tracking shadow probability scores
- **Resonance Analysis:** Monitor biological, pentagonal, and material resonance components
- **Mirror Ly Shadow Tracking:** Track coherence levels and baseline measurements
- **Pictish & Dodecahedron Site Scanning:** Analyze patterns from historical Scottish sites
- **Post-Quantum Cryptography:** ML-DSA-87 verified signatures for all events
- **Interactive Web Interface:** User-friendly dashboard with live metrics

## Technology Stack
- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **Security:** ML-DSA-87 Post-Quantum Signatures
- **Architecture:** Resonance-based detection system
